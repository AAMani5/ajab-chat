## AJAB CHAT

A chat room plugin for Google Chrome.

## Technologies Used
- React: open-source JavaScript library for UI
- Babel: transpiler
- Firebase: database
- Testing - Chai, Mocha, Enzyme, React-Test-Utilities
- Experimented with Google Chrome Extension
